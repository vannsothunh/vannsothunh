# Responsive Personal Portfolio (HTML/CSS/JS)

A clean, accessible, and fast portfolio website featuring:
- Responsive grid layout
- Sticky navigation with mobile menu
- Light/Dark theme toggle (saved to localStorage)
- Intersection Observer entrance animations
- Accessible markup (skip link, proper roles/labels)
- No external dependencies

## Quick Start
1. Unzip the project.
2. Open `index.html` in your browser — you're live!
3. Edit content in `index.html`, styles in `styles.css`, and behavior in `script.js`.
4. Replace placeholders in `assets/` (logo, profile, and project images).

## Customize
- Replace "Your Name" in the header and hero.
- Add your real links (GitHub, LinkedIn, Email).
- Add project screenshots inside `assets/` and update the `<img>` `src` in each card.
- If you have a PDF résumé, drop it into `assets/Your-Name-CV.pdf` (or change the link).

## Deploy
- **GitHub Pages**: push to a repo and enable Pages on the `main` branch.
- **Vercel/Netlify**: drag‑and‑drop the folder or link your repository.
- **Any static host**: upload the files as-is.
